# Design Guide for Begotten Clothing

## Quick Start Design Checklist

### Phase 1: Logo Design
- [ ] Choose design tool (Canva recommended for beginners)
- [ ] Create logo variations (horizontal, vertical, icon-only)
- [ ] Export in multiple formats (PNG, SVG, JPG)
- [ ] Test logo on dark and light backgrounds
- [ ] Get feedback from friends/family

### Phase 2: Product Designs

#### T-Shirt Designs
- [ ] Design front graphic
- [ ] Design back graphic (optional)
- [ ] Consider sleeve designs (optional)
- [ ] Create mockups to visualize
- [ ] Test design at different sizes

#### Hoodie Designs
- [ ] Adapt t-shirt designs for hoodie
- [ ] Consider hood/zipper placement
- [ ] Test design visibility on hoodie material
- [ ] Create mockups

### Phase 3: Product Photography
- [ ] Take photos of products (or use mockup tools)
- [ ] Ensure good lighting
- [ ] Show multiple angles
- [ ] Include size/model information
- [ ] Optimize images for web (compress)

## Design Specifications

### Logo Requirements
- **Format**: SVG (vector) preferred, PNG for web
- **Size**: Should work at 16x16px to 2000x2000px
- **Colors**: Create versions for light/dark backgrounds
- **Transparency**: PNG with transparent background

### Product Design Specifications

#### For Print-on-Demand (DTG)
- **Resolution**: 300 DPI minimum
- **Format**: PNG or JPG
- **Color Mode**: RGB
- **File Size**: Under 25MB typically
- **Dimensions**: 
  - T-Shirt: 4500 x 5400px (15" x 18" at 300 DPI)
  - Hoodie: 4500 x 5400px (15" x 18" at 300 DPI)

#### For Screen Printing
- **Colors**: Limit to 1-4 colors for cost efficiency
- **Format**: Vector (AI, EPS) preferred
- **Separations**: Each color on separate layer

## Quick Design Tips

1. **Keep it Simple**: Simple designs often work best on clothing
2. **Test at Size**: View your design at actual print size
3. **Consider Placement**: Where will the design sit on the garment?
4. **Color Contrast**: Ensure text/graphics are readable
5. **Brand Consistency**: Use consistent colors/fonts across all designs
6. **Get Feedback**: Show designs to others before printing
7. **Start Small**: Test with a few designs before expanding

## Next Steps After Design

1. **Get Samples**: Order 1-2 samples before bulk production
2. **Photograph**: Take professional photos or use mockups
3. **Upload to Shopify**: Add products with images
4. **Update Website**: Replace placeholder images with real designs
5. **Launch**: Start selling!

---

**Remember**: Your design doesn't have to be perfect from day one. Start with something you like, get feedback, and iterate. The most important thing is to get started!
