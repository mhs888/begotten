# Purpose Page Scrolling Images

Place your square images here for the scrolling band at the top of the Purpose page.

## Instructions:

1. Add your images to this folder (`/public/purpose-scroll/`)
2. Name them: `image1.jpg`, `image2.jpg`, `image3.jpg`, etc.
   - Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
   - Recommended size: 400x400px or larger (square format works best)
3. Update the image list in `/app/purpose/page.tsx` if you use different names

## Example:
- `/public/purpose-scroll/image1.jpg`
- `/public/purpose-scroll/image2.jpg`
- `/public/purpose-scroll/image3.jpg`
- etc.

The images will automatically appear in the scrolling band at the top of the Purpose page.
