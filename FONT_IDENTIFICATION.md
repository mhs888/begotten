# Font Identification Guide

## Current Setup

I've updated the site to use **Montserrat** for headings and display text, which is a bold, modern sans-serif that should match your logo aesthetic.

## If You Want to Identify the Exact Font

### Option 1: Online Font Identification Tools

1. **WhatTheFont** (Recommended)
   - Go to: https://www.whatfontis.com/
   - Upload your logo image
   - Select the "BEGOTTEN" text area
   - It will identify the font or suggest similar ones

2. **Fontspring Matcherator**
   - Go to: https://www.fontspring.com/matcherator
   - Upload your logo image
   - Select the text area
   - Get font matches

3. **Adobe Fonts Match Font** (if you have Photoshop)
   - Open image in Photoshop
   - Select text with Rectangular Marquee Tool
   - Go to `Type > Match Font`
   - Photoshop suggests similar fonts

### Option 2: Common Bold Sans-Serif Fonts

If the tools don't find an exact match, these fonts are commonly used for bold logos:

- **Montserrat** (currently using) - Modern, geometric
- **Poppins** - Rounded, friendly
- **Raleway** - Elegant, clean
- **Work Sans** - Professional, versatile
- **Outfit** - Modern, bold
- **Space Grotesk** - Geometric, unique
- **Inter** (body text) - Clean, readable
